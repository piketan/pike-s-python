#python转义
    \\反斜杠
    \'单引号   #如果需要在被单引号包裹的字符串使用单引号就需要在前面加一个反斜杠
    \"双引号   #如果需要在被双引号包裹的字符串使用双引号就需要在前面加一个反斜杠
    \aASCII响铃符
    \bASCII推格符
    \fASCII
    \nASCII换行符
    \N{name}数据库中的字符名name是它的名字
    \rASCII回车符
    \tASCII水平制表符
    \uxxxx值为16位十六进制xxxx的字符串
    \Uxxxxxxxx值为32位十六进制xxxxxxxx的字符
    \vASCII垂直制表符（VT）
    \ooo值为八进制的ooo的字符
    \xhh值为十六进制hh的字符
