#python目前为止学习到的符号
    print()：打印
    #：注释
    变量
    f:字符串前加f格式化
    format：格式化
    end=''：两段print不换行
    \n：换行
    """ """:行内换行
    input:输入
    from import:从库里拿出
    sys：一个库
    argv：sys里面的模块
    script：argv里面的参数可以显示当前文件的名称路径
    open:打开文件
    read:以只读打开文件
    readline：只读一行
    seek()：将指针移动到文件开头
    truncate：清空文件里的内容
    write：改写文件的内容
    close：保存并关闭文件
    exists：判断文件是否存在
    def：定义一个函数
    add()：将括号里的内容相加
    subtract():将文件里的内容相减
    multiply():将文件里的内容相乘
    divide():将文件里的内容相除
